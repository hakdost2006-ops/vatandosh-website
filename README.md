# Vatandosh Connect Website

Official website for Vatandosh Connect, a 501(c)(3) nonprofit connecting global expertise with community development initiatives in Uzbekistan.

This is a static website that can be hosted on Cloudflare Pages, Netlify, GitHub Pages, or any standard static hosting service.

## Pages

- `index.html`
- `about.html`
- `our-work.html`
- `transcaspian-corridor.html`
- `insights.html`
- `transparency.html`
- `privacy-policy.html`
- `contact.html`
- `donate.html`

## Local Preview

```bash
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173/index.html
```
